# Design System Master

## Brand Direction
- Product feeling: private, approachable, vivid, trustworthy
- Trust cues: concise privacy messaging, friendly gradients, strong iconography, low-friction launcher
- Avoid: oversized empty hero space, dense sidebars, duplicate tool surfacing, backend-oriented language

## Color Tokens
- background: blue / cyan / purple vertical gradient, dark graphite gradient in dark mode
- surface: clean white cards, smoky dark panels
- text-primary: charcoal / white
- text-secondary: cool gray
- accent: orange-amber in light mode, purple-blue in dark mode
- border: soft gray

## Typography
- display: bold sans headline
- body: neutral sans for all UI
- body copy stays short, functional, and one sentence max on homepage cards

## Spacing / Shape
- generous outer spacing
- rounded large panels only when they structure the page
- no decorative card stacking

## Components
- header: animated sticky bar with compact logo, top links, theme toggle
- hero: centered badge, product name, privacy line, feature badges
- launcher: search + category tabs + tool card grid with names only, no helper subtitles
- upload zone: oversized shared drop target with one clear heading, no explanatory body copy
- footer strip: privacy block and utility links

## Motion
- entrance slides for header, hero, and controls
- hover lift on tabs and cards
- light tap feedback only
