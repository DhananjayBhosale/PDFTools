# PDF Chef Homepage Redesign

## Goal
Replace the current text-heavy ZenPDF landing page with a simpler, classier PDF Chef homepage that feels private, direct, and easy to scan.

## Approved Direction
- Brand: PDF Chef
- Tone: privacy-focused, minimal, no tool explanations
- Structure: one dominant hero, one compact launcher, one minimal trust/footer strip
- Motion: restrained entrance and hover motion only

## Layout
1. Slim header with brand, source/privacy links, and theme toggle
2. Full-bleed hero with:
   - PDF Chef brand
   - short privacy line
   - short support line
   - strong visual anchor
3. Integrated launcher panel with search and grouped tool names only
4. Minimal footer strip with privacy cue and links

## Content Rules
- Keep copy short enough to scan in seconds
- Do not explain how tools work on the homepage
- Use basic tool names only
- Keep privacy language present but brief

## Visual System
- Warm light surface, dark text, cyan accent
- Serif display for brand moments, clean sans for UI
- Low-chrome panels, subtle atmospheric background, no card mosaic

## Tool IA
- Arrange
- Convert
- Review & Secure
